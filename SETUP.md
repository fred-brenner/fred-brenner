# Setup

1. Create a public GitHub repository named `fred-brenner`.
2. Push this folder to `fred-brenner/fred-brenner`.
3. Create a GitHub token for README stats and add it as a repository secret named `README_STATS_TOKEN`.
4. Run the `Update README cards` workflow once from the Actions tab.

The profile README appears on your GitHub profile when the repository is public and contains `README.md` at the repository root.
